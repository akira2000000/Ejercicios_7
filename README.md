# Ejercicios_7
# Angélica Trejo Gálvez
# Licenciatura en Artes y Comunicación Digitales

El objetivo principal de este repositorio es reunir, organizar y documentar el proceso de trabajo realizado a lo largo de la UEA, ofreciendo un espacio centralizado donde se concentran todos los ejercicios, proyectos y materiales de referencia utilizados durante el curso. Este repositorio busca funcionar como un archivo digital estructurado que permita dar seguimiento al desarrollo de habilidades técnicas y conceptuales adquiridas, así como registrar las fases de producción de cada actividad. De esta forma, sirve tanto como evidencia académica como una herramienta de consulta futura, ya que facilita revisar versiones anteriores, comparar avances y comprender mejor la evolución de cada proyecto.

Además, el repositorio funciona como un archivo que registra el proceso de experimentación: desde entender la estructura básica de un sketch en p5.js, hasta construir funciones que interactúan con tablas, autenticación y APIs de Supabase. El propósito es que este repositorio sea útil no solo durante el curso, sino también en futuros proyectos de arte digital que necesiten recopilar información de usuarios, generar sistemas interactivos basados en datos o construir piezas conectadas a la nube.
